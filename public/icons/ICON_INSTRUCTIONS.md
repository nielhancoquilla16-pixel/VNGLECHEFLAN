# App Icons Instructions

## Required Icon Sizes

Place your app icons in this folder with these exact filenames:

- `icon-72x72.png` (72×72 pixels)
- `icon-96x96.png` (96×96 pixels)
- `icon-128x128.png` (128×128 pixels)
- `icon-144x144.png` (144×144 pixels)
- `icon-152x152.png` (152×152 pixels)
- `icon-192x192.png` (192×192 pixels)
- `icon-384x384.png` (384×384 pixels)
- `icon-512x512.png` (512×512 pixels)

## Design Guidelines

### Icon Design:
- Use the V&G LecheFlan logo or branding
- Background: Yellow (#FBBF24) to match theme
- Include the red "V&G" circle or dessert illustration
- Keep design simple and recognizable at small sizes
- Avoid text that's too small to read

### Quick Generation:
1. Create a 512×512px PNG with your design
2. Use online tools to generate all sizes:
   - https://realfavicongenerator.net/
   - https://www.pwabuilder.com/
   - https://favicon.io/

### Example Design:
```
512×512px canvas
Background: #FBBF24 (yellow)
Center: Red circle (#DC2626) with "V&G" text
Add: Small leche flan illustration or fork icon
```

## Using Figma/Photoshop:

1. Create 512×512px artboard
2. Design your icon
3. Export at 1x, 2x for all required sizes
4. Name files exactly as listed above

## Temporary Placeholder:
Until you create custom icons, the app will use generic placeholders.
For production, create proper branded icons following the guidelines above.
